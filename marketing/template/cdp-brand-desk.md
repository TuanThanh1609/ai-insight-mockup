| `LOGO CDP MOBILE 2.png` | 2084 × 2084 | 1:1 vuông | Variant app icon (dự phòng) |

### Icon Mark (CDP Circle)

- Vòng tròn đồng tâm — dữ liệu đa kênh hội tụ
- Chữ **CDP** cách điệu bên trong
- C + P kết nối liền mạch, D nổi bật thanh ngang
- White trên nền trong suốt → đặt được mọi nền

### Quick Rules

```
✓ TRẮNG.png → nền #1A2138 (Deep Navy) hoặc tối hơn
✓ XANH.png → nền #FCF8FB (Surface) hoặc trắng
✓ MOBILE 1/2 → app icon, avatar, favicon

✗ KHÔNG co giãn / bóp méo
✗ KHÔNG thêm shadow/glow/effect lên logo
✗ KHÔNG đặt XANH.png trên nền tối
✗ KHÔNG đặt TRẮNG.png trên nền nhạt (dùng XANH)
✗ KHÔNG đặt logo trên nền có pattern phức tạp
```

### Clear Space

- Logo ngang: ≥ ½ chiều cao logo — cả 4 cạnh
- Logo vuông: ≥ ½ chiều rộng — cả 4 cạnh

### Minimum Sizes

| Ngữ cảnh | Width tối thiểu |
|----------|----------------|
| Social media | 120px |
| Pitch deck | 200px |
| Landing page | 160px |
| Email header | 100px |
| App icon | 1024×1024px (native) |

### Primary Palette

```

```
#059669  GREEN    → Success, điểm TỐT (≥7.5), trending up
#D97706  AMBER   → Warning, điểm TRUNG BÌNH (3.1–6.9)
#DC2626  RED     → Danger, điểm THẤP (<3), errors, junk lead
#4F81BD  MUTED   → Lead Lạnh (temperature)

```

### ROAS Color Scale

```
≥ 3×    #059669  GREEN   → Tốt — scale được
≥ 1.5×  #D97706  AMBER   → Cải thiện — theo dõi
< 1.5×  #DC2626  RED     → Thấp — cần xem xét
```

### Disease Severity

```
0–3     #BF3003  DEEP RUST   → Nặng
3.1–6   #0052FF  VIBRANT BLUE → Trung bình
6.1–10  #059669  GREEN        → Nhẹ
```

### Lead Temperature

```
Nóng    #DC2626  RED
Ấm     #F79646  ORANGE
Lạnh   #4F81BD  MUTED BLUE
```

### Chart Colors (multi-series)

```
Series 1  #0052FF  Vibrant Blue
Series 2  #BF3003  Deep Rust
Series 3  #059669  Green
Series 4  #D97706  Amber
```

### Color DON'Ts

```
✗ Pure Black #000000 cho text      → dùng #1b1b1d
✗ Pure White #FFFFFF trên Navy     → dùng #fcf8fb
✗ Deep Rust làm nền rộng          → chỉ cho CTA / alert band
✗ Gradient đa màu sặc sỡ          → tối đa 2 màu (navy→blue OK)

## 6. Spacing & Layout

### 8px Grid

```
4px   → icon-to-text gap, dense data tables
8px   → default component spacing
16px  → card padding, button gap
24px  → section internal padding
32px  → card main padding
48px  → section gap
64px  → major section break
96px  → hero section padding
128px → page-level major separation
```

### Border Radius

```
8px   → DEFAULT — tất cả cards, buttons, inputs (KHÔNG dùng 4px)
4px   → ONLY for small inputs, tiny chips
12px  → large modals
16px  → glass panels, very large cards
9999px → pills, avatars
```

### No-Line Rule

```
✗ KHÔNG dùng 1px solid borders cho sectioning
✓ Dùng surface shifts (light → slightly darker)
✓ Dùng khoảng trắng (8–24px) để phân cách
✓ Dùng ghost border nếu bắt buộc: outline rgba(27,27,29,0.08)
```

### Elevation (Ambient Shadow)

```
Shadow color: rgba(27,27,29,0.08) — KHÔNG dùng pure black
Shadow blur:  32px–48px
Shadow opacity: 4%–8%

shadow-sm  → 0 2px 8px  rgba(27,27,29,0.06)
shadow-md  → 0 4px 16px rgba(27,27,29,0.08)
shadow-lg  → 0 8px 32px rgba(27,27,29,0.08)
shadow-xl  → 0 16px 48px rgba(27,27,29,0.12)
```

### Glassmorphism

```
background:  rgba(252,248,251,0.8)   /* surface 80% opacity */
backdrop-filter: blur(12px)
border: 1px solid rgba(27,27,29,0.08)
border-radius: 8px
```

### Responsive Breakpoints

```
Mobile  → < 640px   (1 column)
Tablet  → 640–1024px (2 columns)
Desktop → 1024–1440px (3 columns + sidebar)
Wide    → > 1440px  (max-width container, centered)
```

### Content Width

```
Body text        → 720px  max, centered
Dashboard        → 1440px full-width
Landing hero    → 1200px centered
Marketing images → 1080px native
Slide content    → 1920px
```

## 8. Voice & Copy

### Voice Position

```
AUTHORITY + CALM + FORMALITY + CLARITY
```

### Voice by Touchpoint

| Kênh | Tone | Ví dụ |
|-------|------|--------|
| Landing page | Authority + Calm | "Phân tích 10 nhóm vấn đề. Không phải 1 lần — liên tục cải thiện." |
| Dashboard | Clarity + Calm | "Điểm sức khỏe: 6.2/10 — Cần cải thiện" |
| Facebook ad | Authority + mild Urgency | "Junk Lead 42%? Biết ngay bệnh nào đang kéo tụt." |
| Pitch deck | Authority + Formal | "Social & AI CDP — Giải pháp đầu tiên tại Việt Nam" |
| Email nurture | Friendly + Calm | "Hồ sơ bệnh án tuần này: điểm Lead Quality ↑ 0.3" |
| In-app | Friendly + Clarity | "Chào bạn! Hôm nay mình khám gì?" |

### Headline Patterns

```
PATTERN 1 — Problem-Aware:
  "[Pain Point cụ thể]? [Solution]"
  VD: "Junk Lead 42%? Biết ngay nhóm nào đang kéo tụt."

PATTERN 2 — Transformation:
  "[Before] → [After]"
  VD: "Từ 'không biết' → 'biết chính xác 10 nhóm bệnh'"

PATTERN 3 — Authority Claim:
  "[Số liệu ấn tượng] — [Context]"
  VD: "10 nhóm bệnh. Real-time mỗi ngày. Hành động cụ thể."

PATTERN 4 — Question-Driven:
  "[Câu hỏi] — [Câu trả lời]"
  VD: "Hội thoại khách hàng đang 'bệnh' ở đâu?"
```

### Từ Vựng Chuẩn

```
✓ DÙNG:
  "Khám Bệnh Hội Thoại"     → tên tính năng đầy đủ
  "Hồ sơ bệnh án"           → patient record metaphor
  "Chuyên gia Smax"          → expert agent
  "Lead Nóng / Ấm / Lạnh"  → temperature (KHÔNG Hot/Warm/Cold)
  "Nhóm bệnh" / "Hạng mục" → disease group
  "Gợi ý" / "Hành động"    → recommendation
  "Điểm sức khỏe"           → health score
  "Real-time mỗi ngày"       → daily update
  "Cần cải thiện"           → severity label

✗ KHÔNG DÙNG:
  "Bệnh" (một mình)          → "nhóm bệnh" hoặc "hạng mục"
  "AI" khi chỉ Smax         → "Chuyên gia Smax"
  "Bot" / "Robot"           → "Chuyên gia Smax"
  "Hot/Warm/Cold" (English) → "Nóng / Ấm / Lạnh"
  "Tốt nhất thị trường"     → overpromise
  "Hoàn toàn miễn phí!"     → dùng "Miễn phí"
  Emoji spam (>3/post)      → tối đa 1-2 emoji
```

## 11. DO / DON'T — Tra Cứu Nhanh

### Logo

```
✓ Dùng logo chuẩn — giữ tỉ lệ pixel
✓ TRẮNG → nền tối | XANH → nền sáng
✓ Giữ clear space ½ chiều cao
✗ Không co giãn, thêm shadow, đổi màu
✗ Không tách icon khỏi wordmark
```

### Màu Sắc

```
✓ Deep Navy sidebar + Vibrant Blue links
✓ Deep Rust CTA button (sparingly — laser pointer)
✓ Surface off-white (#FCF8FB) background
✓ Semantic colors đúng context
✗ Pure Black #000 cho text
✗ Deep Rust làm nền rộng
✗ Gradient đa màu sặc sỡ
```

### Typography

```
✓ Manrope Bold headlines + Inter body
✓ Label = ALL CAPS + 5% letter-spacing
✓ Font tiếng Việt đầy đủ (Manrope/Inter/Be Vietnam)
✓ Specific numbers (42%, 23 phút, 6.2/10)
✗ Arial/Helvetica làm display font
✗ Courier/monospace cho marketing copy
```

### Imagery

```
✓ Real product screenshots
✓ Vietnamese business context
✓ Data visualization
✓ Warm tones photography
✗ Stock photo families
✗ Western office environments
✗ Abstract 3D renders
```

### Layout

```
✓ 8px grid system
✓ 8px border-radius (DEFAULT)
✓ Surface shifts thay vì border lines
✓ Tinted ambient shadows
✗ 1px solid borders
✗ 4px border-radius (chỉ cho inputs nhỏ)
✗ Pure black shadows
```

### Voice & Copy

```
✓ Vietnamese language throughout
✓ Specific data (42%, 2 phút, 6.2/10)
✓ Empathetic — thấu hiểu pain point
✓ Action-oriented — luôn có solution
✗ Generic claims ("tốt nhất", "hoàn toàn")
✗ English jargon khi có tiếng Việt chuẩn
✗ Emoji spam (>3 per post)
```
